# YUMI APP

## Cara build APK via GitHub Actions

1. Push kode ke branch `main`.
2. GitHub akan otomatis build APK release.
3. Setelah build selesai, buka tab **Actions** di repo.
4. Pilih workflow terbaru, cari bagian **Artifacts**.
5. Download file `yumi-app-release-apk.zip`.
6. Ekstrak dan install APK di Android device.
